## 2001 : 최소 대금 

```python
pasta=2000
juice=2000
for i in range(3): #3번째 까지 pasta
  num=int(input())
  if num<=pasta:
    pasta=num
for i in range(2): #2번째 까지 juice
  num=int(input())
  if num<=juice:
    juice=num
print(round((pasta+juice)*1.1,2))
```
