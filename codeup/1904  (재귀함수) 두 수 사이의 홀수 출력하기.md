## 1904  (재귀함수) 두 수 사이의 홀수 출력하기 

```python
a,b=map(int,input().split())

def odd(num,end_num):
  if num%2!=0:
    print(num,end=' ')
  if num==end_num:
      return
  return odd(num+1,end_num)
odd(a,b)  # print 시에 None 이 나오니 주의 ㅜㅜ
```
