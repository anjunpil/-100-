## 1807  우박수 길이 (3n+1) (small)

```python
a,b=map(int,input().split())

num=0
Max_num=0
for i in range(a,b+1):
  count=0
  n=i
  while True:
    if n==1:
      count+=1
      break
    if n%2==0:
      n=n/2
      count+=1
    elif n%2!=0:
      n=3*n+1
      count+=1
  if num<count:  #num값에다 메모제이션이용
    num=count
    Max_num=i
print(Max_num,num)

```
