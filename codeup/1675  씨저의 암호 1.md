## 1675  씨저의 암호 1

```python
num=input().split()
secret_num=[]
for i in range(len(num)):
  secret=''
  for j in num[i]:
    if j=='a':  	# ord함수를 이용, a,b,c의 값을 chr 치환시 기호로
      secret+='x'	# 넘어가 버려서 a,b,c 설정만 따로 해줌
    elif j=='b':
      secret+='y'
    elif j=='c':
      secret+='z' 
    else:
      secret += chr(ord(j)-3)
    
  secret_num.append(secret)
print(*secret_num)
```
